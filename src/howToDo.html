How To Do

Welcome To Rapidbook.in. First Of All Thanks For Selecting This Software. This Software is all about Tatakal ticket booking making the life of common man easier. How many times you tried to book tatkal ticket from irctc and how many times you succeed. Well It's very hard. Many Times you stuck waiting for your ticket to confirm. Well here is the solution for you. You can book your tatkal ticket using Rapidbook.in as fast as you can.... So let's start our How To Do guide...

Steps Required:

  1. Since You already on this page this means that you already opened Rapidbook.in

  2. So go to Rapidbook.in and click Go To Tatkal Form


  3. After that you will be transferred to autofill form Now if you not already installed plugin you can see install plugin button as below click that button for the form to work properly. In case If you see Plugin Installed button in place of Install Plugin button The Go to step 6 directly.


  4. After clicking install plugin wait for the below popup to come and then click add extension


  5. After The extension is installed Reload the page and now you see Plugin Installed button in place of Install Plugin button if you see this you are good to go.


  6. Now Fill The Username And Password as Provided By Irctc For Login To Irctc


  7. Fill The train Details, date Class etc


  8. Now Fill The Passengers Details Only Four Passenges are allowed at the tatkal time


  9. Sele Book Tickets If confirm birth alloted and give Mobile details you want the ticket details will be send on


  10. Enter child passengers Details if any


  11. Now Most Important Give Payment Details


  12. Make Sure You Selected The Tatkal Quota


  13. After You Have Filled All The Details It's Time To Go on And Click save Button then Go To IRCTC Website To Book Ticket


  14. Thanks For Using Rapidbook.in
